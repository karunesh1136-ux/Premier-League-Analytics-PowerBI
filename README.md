# Premier-League-Analytics-PowerBI

Overview

Interactive Power BI dashboard analyzing 12 seasons of EPL data. Designed to visualize team performance trends and defensive/offensive metrics through a professional UI.

Key Features

Dynamic Trends: Line chart with tooltips showing Wins/Losses on hover.

Smart Filtering: Searchable team slicer and horizontal season tiles.

Visual Rankings: Standings table with conditional data bars for quick comparison.

Mobile Ready: Dedicated UI layout optimized for mobile devices.

KPI Tracking: High-level summary of Goals, Cards, and Clean Sheets.

Technical Skills

DAX: Calculated measures for league-wide totals.

UI/UX: Applied visual hierarchy and consistent color branding.

Data Modeling: Transformed raw CSVs into a relational schema.
